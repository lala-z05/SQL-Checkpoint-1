# SQL Verilənlər Bazası Layihəsi
## Layihə haqqında
Bu layihə Northwind verilənlər bazası üzərində SQL sorğularının yazılması və məlumatların təhlili məqsədilə hazırlanmışdır.
## İcra olunan mövzular
- SELECT, WHERE, ORDER BY
- INNER JOIN və LEFT JOIN
- GROUP BY və HAVING
- Subquery (Alt sorğular)
- CTE (WITH)
- Window Functions (ROW_NUMBER, RANK, SUM() OVER)
- Sorğuların optimallaşdırılması
- İndeks (Index) anlayışı
